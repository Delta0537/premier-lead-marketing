# lovabel.dev — Enterprise Vibe Coding Framework

**PLM's central hub for vibe coding.**  
English as programming language. Cursor (core) + Lovable/v0/Replit (scaffold) + Claude Code CLI (polish) + GHL.

## What's here

- **Stack** — Cursor, Lovable, v0, Replit, Claude Code CLI
- **Enterprise workflow** — Scaffold → Cursor → Refine → Agentic polish
- **Get started** — First prompt to paste into Cursor Composer
- **Quick links** — All tools + GHL, GitHub, Vercel
- **Framework** — Full vision doc (framework.html)
- **SOP** — GHL use cases, landing pages, client portals (sop.html)

## Run locally

```bash
npm start
# → http://localhost:3333
```
