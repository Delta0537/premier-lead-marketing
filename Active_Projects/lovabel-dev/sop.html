STANDARD OPERATING PROCEDURE (SOP)
AI "Vibe Coding" Stack

Effective: January 2026 | Owner: PLM | Stack: Lovable.dev (UI) + Cursor (Logic/Refinement)

1. Executive Summary

This SOP defines the workflow for using Generative AI to build enterprise-grade websites, landing pages, and interactive marketing assets. By bypassing traditional website builders (WordPress/ClickFunnels), PLM retains full code ownership, increases site speed, and eliminates monthly licensing fees.

  • Lovable.dev: Rapid visual prototyping, UI design, initial codebase. (The Architect)
  • Cursor: Refining code, connecting APIs (GoHighLevel), fixing bugs, deployment. (The Builder)

2. Prerequisites & Setup

  • Lovable Account — For generating the initial project
  • GitHub Account — To host repositories
  • Cursor Editor — Installed (VS Code fork)
  • Node.js — For running code locally
  • Supabase (Optional) — For databases, user logins

3. Use Case 1: Agency Websites

Goal: High-end PLM-style brochure site.

Step 1 (Lovable)

"Build a high-end digital agency website. Theme: Black, Gold, White. Typography: Montserrat and Open Sans. Sections: Hero with 'Vibe Coding' hook, Services Grid, 'Our Process' timeline, Case Studies with data viz, 'Book Strategy' footer."

Refine: "Make the gold gradient softer," or "Add glassmorphism to service cards."

Step 2 (Hand-off)

Export to GitHub in Lovable → Clone in Cursor.

Step 3 (Cursor)

  • Mobile: "Check responsive design. Ensure Hero text doesn't overlap."
  • SEO: "Add meta tags, title 'Premier Lead Marketing | Mobile AL', OpenGraph."
  • Animations: "Add Framer Motion fade-in to service cards on scroll."

4. Use Case 2: Landing Pages (GHL Funnels)

Goal: Real estate capture page (e.g. "What's My Home Worth?").

Step 1 (Lovable)

"Create a single-page real estate landing page. Hero: 'Mobile, AL Real Estate Market Report - Free Download.' Center: Form for Address + Email. Below fold: 3 bullets on why market is booming (Airport, Port). Style: Clean, trustworthy, blue and white."

Step 2 (Cursor — GHL Integration)

"Connect this form to a webhook. On submit, send Address and Email to this GHL Webhook URL: [Insert URL]. Add error handling. After success, redirect to Thank You page."

5. Use Case 3: Link-in-Bio / Calculators

Strategy A: Link Tree style

"Build a mobile-first Link Tree style page. Profile picture top. 5 glassmorphism buttons: 'Home Valuation,' 'Search Homes,' 'Partner with Us.' Background: subtle animated black/gold gradient."

Strategy B: Viral Calculator

"Build a Rent vs Buy calculator. Inputs: Monthly Rent, Home Price. Output: Chart showing net worth over 10 years. Chart animates."

You own traffic data, not Linktree. Screen-record for TikTok.

6. Use Case 4: Client Portals (CEO Dashboard)

Step 1 (Lovable)

"Build a SaaS dashboard. Sidebar: Leads, Pipeline Value, ROI. Main: 3 metric cards, line graph for lead growth. Dark mode, enterprise style."

Step 2 (Cursor)

  • "Connect charts to mock JSON for now. Structure so we can swap with GHL API later."
  • "Wrap dashboard in Supabase auth so only logged-in users see it."

7. Deployment

  1. Create free Netlify or Vercel account
  2. Connect GitHub repository
  3. Click Publish
  4. Add custom domain (e.g. portal.premierleadmarketing.ai)

8. Micro-Test

Start with Use Case 2 (Landing Page) — fastest path to ROI. Use the "Mobile AL Real Estate Market Report" prompt above.
