Vibe Coding Framework

PLM vision for enterprise-grade sites using AI-native tools. English as programming language.

What is vibe coding?

Prompting AI to build significant chunks of functionality (or entire apps) using natural language. You treat English as the programming language. The stack centers on Anthropic’s Claude 3.5 Sonnet.

1. Core editor: Cursor

Primary tool for enterprise-grade code. VS Code fork with AI built in.

  • Model: Claude 3.5 Sonnet (king of coding tasks)
  • Composer (Cmd+I): Type a prompt → generates multiple files, edits existing code, structures the project in real time
  • Enterprise: Runs locally. Works on massive, secure codebases without breaking your workflow
"Build a dashboard with sidebar nav, dark mode toggle, settings page using React and Tailwind."

2. Rapid scaffolders (web-based)

For 0→1 projects. Generate the initial repo in the browser, export to Cursor for fine-tuning.

Lovable (lovable.dev)

  • Best for: Full-stack SaaS with database
  • Describe your SaaS → Lovable builds UI (React/Shadcn), connects Supabase, handles auth
  • Enterprise: Syncs to GitHub. Pull into Cursor for custom logic

v0 (v0.dev by Vercel)

  • Best for: Production-grade UI components
  • Prompt for a complex element (pricing table, tooltips, currency toggle) → copy-paste React + Tailwind
  • Often cleaner and more accessible than other generators

Replit (replit.com)

  • Best for: Instant deploy, collaborative agents
  • "Build a CRM for a dental office" → agent iteratively builds, debugs, deploys
  • Claude Code integrated. Good for teams who don’t want to manage local envs

3. Autonomous agent: Claude Code CLI

Official CLI from Anthropic. Runs in your terminal.

  • Use: "Refactor the auth module to use NextAuth instead of custom JWTs"
  • Explores filesystem, reads files, plans changes, edits code, runs tests
  • Enterprise: Janitorial work, complex refactors across large codebases while you focus on architecture
"Fix all the TypeScript errors in the /app directory."

4. Enterprise workflow

  1. Scaffold with Lovable or v0 → beautiful working UI in ~10 min
  2. Move to Cursor → export to GitHub, open repo
  3. Refine with Claude 3.5 Sonnet → business logic, APIs (Stripe, Salesforce, GHL), security
  4. Agentic polish → Claude Code CLI in terminal for bugs and refactors
