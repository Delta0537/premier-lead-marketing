Enterprise workflow

1 Scaffold — Lovable or v0 → working UI in ~10 min
2 Move to Cursor — Export to GitHub, open repo
3 Refine — Composer: business logic, APIs (GHL, Stripe, Salesforce), security
4 Agentic polish — Claude Code CLI: “Fix all TS errors in /app”

Get started

First step: Cursor + Claude 3.5 Sonnet. Open Cursor → Settings → Models → select Claude 3.5 Sonnet. Then Cmd+I (Composer) and paste:

"Build a simple landing page. Hero: 'PLM Vibe Coding Hub.' Subtext: 'Enterprise-grade sites with AI.' Button: 'Get Started' that scrolls to a short 3-bullet features section. Use Tailwind, clean dark theme."

Ship that, then add GHL webhooks, auth, or deploy to Vercel. See SOP for full prompts.
